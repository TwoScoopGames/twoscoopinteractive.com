# twoscoopinteractive.com



## Setup

`npm install`

Use `npm start` to start the gulp dev server.

Gulp modules setup and what they do on file save:

* `browserSync` - acts like webpack dev server
* `sass` - compile sass
* `minify-css`
* `minify-js`
